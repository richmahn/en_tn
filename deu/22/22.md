# General Information:

Moses speaks to the Israelites as if they were one man, so the word "you" is singular. (See: [[rc://en/ta/man/translate/figs-you]])

# If a man is found

This can be stated in active form. AT: "If someone finds a man" (See: [[rc://en/ta/man/translate/figs-activepassive]])

# and you will remove

"in this way you will remove"
