## translationNotes

* **Merathaim** - This is another name for Babylon. This is a name that means "double rebellion". (See: [[:en:ta:vol1:translate:translate_names]])
* **Pekod** - This is another name for the Chaldeans. This is a name that means "visitation." (See: [[:en:ta:vol1:translate:translate_names]])
* **Put them to the sword** - This refers to having a sword come down on people to kill them. AT: "kill them" (See: [[:en:ta:vol2:translate:figs_synecdoche]])
* **Yahweh's declaration** - See how you translated this in [[en:bible:notes:jer:01:07]].
