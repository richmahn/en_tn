# this word came from Yahweh, saying

This idiom is used to introduce a special message from God. See how you translated a similar phrase in [Jeremiah 18:1](../18/01.md), and make any changes as needed. Alternate translation: "This message came from Yahweh. He said" or "Yahweh spoke this message" (See: [[rc://en/ta/man/translate/figs-idiom]])

# this word came from Yahweh, saying

It can be stated clearly who Yahweh gave this message to. Alternate translation: "this word came to Jeremiah from Yahweh, saying" or "Yahweh spoke this message to Jeremiah" (See: [[rc://en/ta/man/translate/figs-explicit]])
