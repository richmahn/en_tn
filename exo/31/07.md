# tent of meeting

This is another name for the tabernacle. See how you translated this in [Exodus 27:21](../27/21.md).

# ark of the testimony

The ark is the chest that contains the commandments. This can be stated clearly in the translation. See how you translated this in [Exodus 26:33](../26/33.md). Alternate translation: "the chest containing the commandments" (See: [[rc://en/ta/man/translate/figs-explicit]])

# atonement lid

This is the lid that sits on top of the ark where the atonement offering was made. See how you translated this in [Exodus 25:17](../25/17.md).
