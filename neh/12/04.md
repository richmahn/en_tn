# Iddo ... Ginnethon ... Abijah ... Mijamin ... Moadiah ... Bilgah ... Shemaiah ... Joiarib ... Jedaiah ... Sallu ... Amok ... Hilkiah ... Jedaiah ... Jeshua.

These are all names of men. (See: [[rc://en/ta/man/translate/translate-names]])
