# each cubit

A cubit is 46 centimeters. Alternate translation: "every 46 centimeters" (See: [[rc://en/ta/man/translate/translate-bdistance]])

# ten to each cubit

"ten per cubit"

# when the sea itself was cast

This can be stated in active form. Alternate translation: "when the workers cast the sea itself" (See: [[rc://en/ta/man/translate/figs-activepassive]])
