# A good name is to be chosen over great riches

These words can be translated in active form. AT: "A person should choose a good name rather than great riches" (See: [[rc://en/ta/man/translate/figs-activepassive]])

# A good name

"To have others think that one is a good person"

# have this in common

"are alike in one way" or "are the same in this way"
