# General Information:

These verses continue and end the introduction that began in [Proverbs 22:17](./17.md).

# Have I not written for you ... you?

This rhetorical question can be translated as a statement. AT: "You need to know that I have written for you ... you." (See: [[rc://en/ta/man/translate/figs-rquestion]])

# thirty sayings

Some translations read, "excellent sayings." (See: [[rc://en/ta/man/translate/translate-textvariants]])
