# Abaddon

This is a name that means "destroyer." Alternate translation: "the Destroyer" (See: [[rc://en/ta/man/translate/translate-names]])

# are never satisfied

"are never filled up"

# a man's eyes

Here the "eyes" represent a man's desires. Alternate translation: "a man's desires" (See: [[rc://en/ta/man/translate/figs-metonymy]])
