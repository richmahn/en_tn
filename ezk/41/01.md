# General Information:

The man in Ezekiel's vision ([Ezekiel 40:3](../40/03.md)) continues to show Ezekiel around the temple.

# the temple's holy place

the room in the temple in front of the "most holy place"

# six cubits in width on either side

These cubits were the "long" cubits ([Ezekiel 40:5](../40/05.md)), 54 centimeters. See how you translated this in [Ezekiel 40:5](../40/05.md). "they were six cubits in width on both sides"

# six cubits

about 3.2 meters. (See: [[rc://en/ta/man/translate/translate-bdistance]])

# cubits

Each long cubit was about 54 centimeters. See how you translated this in [Ezekiel 40:5](../40/05.md). (See: [[rc://en/ta/man/translate/translate-bdistance]])

# ten cubits

about 5.4 meters (See: [[rc://en/ta/man/translate/translate-bdistance]])

# five cubits

about 2.7 meters (See: [[rc://en/ta/man/translate/translate-bdistance]])

# the dimensions

"the size"

# forty cubits

about 22 meters (See: [[rc://en/ta/man/translate/translate-bdistance]])

# twenty cubits

about 11 meters (See: [[rc://en/ta/man/translate/translate-bdistance]])
