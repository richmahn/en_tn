# If he ... shuts anyone up

"If God ... shuts anyone up in prison"

# if he calls anyone to judgment

The abstract noun "judgment" can be translated with the verb "judge." Alternate translation: "if God calls anyone to go to him so that God might judge him" (See: [[rc://en/ta/man/translate/figs-abstractnouns]])

# who can stop him?

This question emphasizes that no one can stop God. Alternate translation: "no one can stop him!" (See: [[rc://en/ta/man/translate/figs-rquestion]])
