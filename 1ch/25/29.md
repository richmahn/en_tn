# the twenty-second ... the twenty-fourth

"the 22nd ... the 23rd ... the 24th." or "number 22 ... number 23 ... number 24." This shows the order in which the families were chosen. (See: [[rc://en/ta/man/translate/translate-ordinal]] and [[rc://en/ta/man/translate/translate-numbers]])
