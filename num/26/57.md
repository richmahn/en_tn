# General Information:

This is a list of the Levite clans. Moses counts the Levites separately from the other tribes because they did not receive any land. (See: [[rc://en/ta/man/translate/translate-names]])

# counted clan by clan

This can be stated in active form. Alternate translation: "that the leaders also counted clan by clan" (See: [[rc://en/ta/man/translate/figs-activepassive]])
