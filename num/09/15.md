# the tabernacle was set up

This can be stated in active form. Alternate translation: "the Levites set up the tabernacle" (See: [[rc://en/ta/man/translate/figs-activepassive]])

# the tent of the covenant decrees

This is another name for the tabernacle. See how you translated the phrase "the tabernacle of the covenant decrees" in [Numbers 1:50](../01/50.md).

# It appeared like fire until morning

This refers to the cloud's appearance during the night. Here the cloud is compared to looking like a fire. Alternate translation: "During the night the cloud looked like a huge fire until morning" (See: [[rc://en/ta/man/translate/figs-simile]])
