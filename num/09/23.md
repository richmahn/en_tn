# at Yahweh's command

The word "command" can be expressed as a verb. Alternate translation: "what Yahweh commanded" (See: [[rc://en/ta/man/translate/figs-abstractnouns]])

# Yahweh's command given through Moses

This can be stated in active form. Alternate translation: "the command that Yahweh had given through Moses" (See: [[rc://en/ta/man/translate/figs-activepassive]])
